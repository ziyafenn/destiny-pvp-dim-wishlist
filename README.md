# Destiny 2 PVP DIM Wishlist

DIM wishlist of popular PVP weapons based on [light.gg](https://light.gg) loadout analytics (Season 29).

Ranks weapons by usage popularity — **does not** filter for specific perk rolls. Use light.gg's Roll Appraiser for that.

## How to Import

1. Open [DIM](https://app.destinyitemmanager.com)
2. Go to **Settings → Wishlists**
3. Paste this URL:
   ```
   https://raw.githubusercontent.com/ziyafenn/destiny-pvp-dim-wishlist/refs/heads/main/pvp-dim-wishlist-no-perks.txt
   ```

## Contributing

Wishlist has no perk filters — contributions welcome.

- **Via Little Light**: load `pvp-dim-wishlist-no-perks.json` into [Little Light Wishlist Editor](https://wishlists.littlelight.club/), add perks, export, open PR
- **Direct edit**: modify `pvp-dim-wishlist-no-perks.txt` manually per DIM wishlist format
