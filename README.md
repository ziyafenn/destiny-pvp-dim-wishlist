# Destiny 2 PVP DIM Wishlist

DIM wishlist of popular PVP weapons based on [light.gg](https://light.gg) loadout analytics (Season 29).

Ranks weapons by usage popularity — **does not** filter for specific perk rolls. Use light.gg's Roll Appraiser for that.

## Files

| File | Format | Use |
|------|--------|-----|
| `pvp-dim-wishlist-no-perks.txt` | DIM wishlist `.txt` | Import directly into DIM |
| `PVP.json` | JSON | Alternative / programmatic use |

## How to Import

1. Open [DIM](https://app.destinyitemmanager.com)
2. Go to **Settings → Wishlists**
3. Paste the raw URL of `pvp-dim-wishlist-no-perks.txt`
